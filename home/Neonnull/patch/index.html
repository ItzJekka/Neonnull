Patch Notes Pre-Alpha V0.32.1

Release Date: March 29, 2026

Welcome to the latest patch notes for Neon Null! In this update, we've made significant improvements to the game's performance and stability, as well as added new features and content. Here's a breakdown of what's new:

  • New:
    • Added a new tech upgrade, 'Recursive Payload'.
    • Added a new cosmic+ upgrade, 'Gravity Well'.
    • Added a new cosmic+ upgrade, 'Volatile Cells'.
    • Added a new tech upgrade, 'Reactor Breach'.
    • Added a new tech upgrade, 'Adaptive Focus'.
    • Added common, uncommon, rare and epic rarities of an upgrade that reduces rocket recharge time.
    • Added the elite rhombus to the enemy pool.
  • Changes:
    • Added a new enemy death effect, much more fitting and colorful than the original.
    • Added sound, visual and status effects to the cross mine from the new cross enemy.
    • Added tech and cosmic+ upgrades to the titan shop.
    • Adjusted projectile damage on the square boss.
    • Slightly adjusted the size of the projectiles on the square boss.
    • Overhauled the neon border for elite enemies.
    • Touched up the crosshair to give it a cleaner look and also make it consistently sized across all resolutions. Also decreased its size slightly.
    • Split shot angle is now tighter for better performance on bosses. Makes it less of a detrimental pick and more of a useful one.
    • Slightly increased the average gap size in the neon barrier mechanic.
    • Added some additional FX to the titan's wormhole intro / outro sequence.
    • Slightly increased the visual queues for the titan's wormhole event.
    • Removed chromatic aberration visual FX, should result in certain visuals looking less blurry.
    • Adjusted the 2.5D bevel effect to get rid of the bevel dot in the center of most shapes. Future changes coming to remove it from any shapes that still show it.
    • Added bullet prediction to the combat drone.
    • Slightly lowered the rate at which players collect Encrypted Data.
  • Fixes:
    • Vastly adjusted the hitbox of each ship model.
    • Fixed title issues and badge issues on some bosses.
    • Fixed a number of scaling issues on ultrawide monitors.
    • Fixed hitbox issues on the square bosses projectiles.
    • Fixed combat drone not doing damage to the square boss and its turrets.
    • Fixed a bug with the titan's voice spawning voice line.
    • Fixed the titan's eyes being derpy.
    • Fixed the titan's tethers being on top of the shapes comprising it instead of behind them as intended.
    • Fixed a bug where certain bosses would spawn in the same frame as the upgrade menu, causing them to show up in the background. They now initialize AFTER the upgrade.
    • Fixed a bug where the titan event would start in the same frame as the upgrade menu, causing it to be a little buggy. It now initializes AFTER the upgrade.
    • Fixed a bug where projectiles would destroy cross mines.
    • Fixed an issue with one of the menu settings overlapping in the controller section.
    • Balanced a number of sounds that were too loud.
    • Fixed a bug where the player would shift locations while traveling through the titan's wormhole.
    • Fixed a bug where asteroids could hit the player while in stealth (decoy beacon).
    • Fixed several other smaller, insignificant bugs.

Thank you for playing Neon Null! We hope you enjoy the new content and improvements. As always, we welcome your feedback and suggestions for future updates.
