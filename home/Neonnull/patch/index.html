Patch Notes Pre-Alpha V0.30

Welcome to the latest patch notes for Neon Null! In this update, we've made significant improvements to the game's performance and stability, as well as added new features and content. Here's a breakdown of what's new:

  • New:
    • Added a new upgrade type -- Cosmic+
    • Added a new Cosmic+ upgrade - Artillery Protocol
    • Added a new Cosmic+ upgrade - Maintenance Module
    • Added a very early first testing point of our new ship AI companion. The voice line is a placeholder and so is the presentation, a more complete implementation will be available in tomorrow's patch (the 27th).
    • Added a new base model for the ship (no longer a placeholder).
    • Added the ship's visual preview to the hangar.
    • Added multiple new, ship altering designs that are influenced by which drive the player has equipped. Changes the thrusters and the lights on the ship.
    • Added a color picker to the 'color' tab that allows the player to choose which color they want for their hull / ship lights, as well as their boost.
    • Completely overhauled the boosting visual effects of all the ships. I'm still touching this up a LOT, tomorrow it will look a lot better than it does even now. Bare with me on this one please
    • Added the same 3D bevel we use for enemies to the hull shapes to give them a little bit of definition.
    • Added rumble support for controllers. NOTE: Context Buttons will be added for controllers tomorrow, I did add some of it today but not everything that is planned.
  • Changes:
    • Added new sounds to the drone companion and tweaked all of their volume based on distance to be smoother than they were previously.
    • The drone AND the micro-missiles for the drone will now target any enemy you have locked on. I'll end up expanding on this behavior and its controllability in a later patch.
    • Added uncommon, common, rare and epic versions of the thruster upgrade. Lowered the % that they give to balance.
    • Added uncommon and common variants of plasma beam damage.
    • Hulls no longer rotate in the ship's viewport (will probably come back later).
    • Removed the turret from the tip of the ship (this will come back in a different form later).
    • Adjusted the balancing of the drives to give them their own individual profiles, while still making sure more expensive drives perform better.
    • Mark II drive has been replaced with Ghost I drive. Drives will be upgradeable using bits in the very near future.
    • Overhauled the prestige bar to look significantly better than it did previously.
    • Increased the size of the hexagon sprite by approximately 50%, and increased its overall barrier size by 25%.
    • Wormhole event can no longer happen when the player is in the middle of a challenge.
    • Adjusted controller layout to be more user friendly. Mappable controls are coming in an update in early March, so hang in there.
    • Added a 0.5s delay between using decoy beacon and going invulnerable / stealth. Taking damage cancels it without using the charge.
    • Adjusted the glow around the UI elements when playing on controller to be way less obnoxious.
    • Adjusted the projectiles of the blue shooter triangles.
  • Fixes:
    • Fixed a bug with the thruster upgrade that was causing it to not give the player any actual increase.
    • Fixed a bug where graviton pellets could show up in the list of choices from a cosmic gem when the player didn't already have Singularity Sledge.
    • Fixed a bug where Firmware Update could show up if you weren't wearing that neural module (even though there isn't a second one yet).
    • Fixed a bug where Entropy Harvester could show up as an option from the cosmic gem selection AND upgrade menu even if you didn't have a DoT source yet.
    • Fixed plasma beam damage increase not working. Also changed the stat preview and description text to show in damage numbers instead of percentage.
    • Fixed an unintentional mechanic that was causing the tether from the star clone to do damage to the player every 0.5s.
    • Fixed a bug with vampiric rounds that was causing them to not trigger.
    • Controller support is fully updated (there were a bunch of things missing from it since I haven't touched it in awhile).

Thank you for playing Neon Null! We hope you enjoy the new content and improvements. As always, we welcome your feedback and suggestions for future updates.
